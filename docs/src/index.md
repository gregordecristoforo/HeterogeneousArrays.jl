# Welcome to HeterogeneousArrays
HeterogeneousArrays is a Julia package for doing amazing things!
## Installation
```julia
using Pkg
Pkg.add("HeterogeneousArrays")
```

Here's how to use HeterogeneousArrays:

```julia
using HeterogeneousArrays
HeterogeneousArrays.HeterogeneousVector(u = 3.1u"m", v = 5.2u"s")
```
